# An-lise-e-Previs-o-do-Consumo-de-Energia-em-Ve-culos-El-tricos
Desenvolvi um pipeline completo em R para analisar e prever o consumo médio de energia de veículos elétricos, com base em dados extraídos de uma planilha Excel. O projeto envolveu desde a limpeza e transformação dos dados até a construção de modelos estatísticos e de machine learning.
